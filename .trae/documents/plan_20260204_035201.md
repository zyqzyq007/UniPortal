## 目标
根据用户提供的设计规范（图示颜色），统一门户前端的 UI 视觉风格。核心调整是将现有蓝色系/默认色替换为规范中的深藏青色与鲜亮蓝色。

## 视觉规范提炼
根据截图信息：
1.  **主色调 (Primary Action)**：深藏青色
    *   Hex: `#1e3a8a`
    *   应用场景：主要操作按钮、顶部导航栏背景、核心强调区域。
2.  **次要/辅助色 (Secondary Action)**：鲜亮蓝色
    *   Hex: `#2563eb`
    *   应用场景：侧边栏选中态、辅助按钮、图标高亮、链接文字。
3.  **成功/执行色 (Success)**：绿色
    *   Hex: `#16a34a`
    *   应用场景：任务成功状态、执行操作。

## 实施方案

### 1. 定义 CSS 变量
在全局样式文件 `src/styles/global.css` 中定义 CSS 变量，便于统一管理与后续维护。
```css
:root {
  --color-primary-900: #1e3a8a; /* 深藏青 */
  --color-primary-600: #2563eb; /* 鲜亮蓝 */
  --color-success-600: #16a34a; /* 成功绿 */
  /* 辅助灰度保持不变，或根据需要微调 */
}
```

### 2. 全局样式替换
搜索并替换项目中的硬编码颜色值：
*   **顶部导航栏** (`DefaultLayout.vue`)：背景色由 `#0f172a` (Slate-900) 调整为 `--color-primary-900`。
*   **侧边栏** (`DefaultLayout.vue`)：
    *   背景色保持深色或微调为与主色调协调的深色。
    *   选中态背景色由 `#1d4ed8` 调整为 `--color-primary-600`。
*   **按钮** (`global.css` 及各页面)：
    *   `.btn` (主按钮) 背景色调整为 `--color-primary-900`。
    *   `.btn-light` / `.btn-outline` (辅助按钮) 的悬停色或边框色适配 `--color-primary-600`。
*   **链接与高亮**：
    *   文本链接颜色调整为 `--color-primary-600`。
    *   任务“运行中”状态颜色调整为 `--color-primary-600`。

### 3. 组件级调整
*   **ProjectHome.vue**：
    *   快捷入口卡片的图标背景色、Hover 边框色适配新色系。
    *   统计卡片中的“运行中”数字颜色。
*   **ToolsCenter.vue**：
    *   “进入工具”按钮应用主色调。
*   **ResultView / TaskDetail**：
    *   状态标签颜色适配。

### 4. 验证与微调
*   检查所有页面（登录、列表、概览、工具、任务），确保颜色替换无遗漏且视觉和谐。
*   确保文字与背景的对比度符合可读性要求（特别是深色背景下的白字）。

## 执行计划
1.  **更新全局 CSS**：在 `global.css` 中定义变量并替换通用组件（按钮、链接）样式。
2.  **调整布局颜色**：修改 `DefaultLayout.vue` 中的顶栏与侧栏颜色。
3.  **适配页面细节**：逐个检查并调整 `ProjectHome.vue`、`ToolsCenter.vue` 等页面的局部颜色定义。
