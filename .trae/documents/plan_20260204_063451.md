## 问题分析
用户指出登录页和上传页的“标签（文字）”与“输入框（窗口）”距离太近。
经检查，`Login.vue` 和 `ProjectsUpload.vue` 均使用了全局样式类 `.form`。
HTML 结构均为：
```html
<form class="form">
  <div>
    <label>...</label>
    <input ...>
  </div>
  ...
</form>
```
目前 `.form` 定义在 `src/styles/global.css` 中：
```css
.form {
  display: flex;
  flex-direction: column;
  gap: 12px;
}
.form label {
  font-weight: 600;
  font-size: 14px;
}
```
问题在于 `label` 是行内元素（默认）或未设置下边距，而它与 input 紧挨着。虽然 `.form` 设置了 `gap: 12px`，但这控制的是直接子元素（即包裹 label 和 input 的那个 `div`）之间的间距，**并没有控制 label 和 input 内部之间的间距**。

## 修复方案
在全局样式 `src/styles/global.css` 中，为 `.form label` 增加 `margin-bottom` 属性，并将其显示模式改为 `block` 或 `inline-block` 以便 margin 生效。这将统一修复所有使用 `.form` 类的页面（包括登录页、上传页等）。

### 实施步骤
1.  **修改** `src/styles/global.css`：
    *   在 `.form label` 选择器中添加 `display: block;` 和 `margin-bottom: 8px;`。
    *   同时检查 `.form` 下的 `div` 容器，可能需要微调间距以保持整体协调。

## 预期效果
*   登录页的“账号”、“密码”文字下方将增加 8px 间距。
*   上传页的“项目名称”、“项目描述”等文字下方也将增加 8px 间距。
*   整体表单布局更加舒展，提升可读性。
