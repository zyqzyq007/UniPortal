# Implement Delete Project Functionality

I will implement the functionality to delete projects directly from the project list view.

## 1. UI Implementation in `ProjectsList.vue`
-   **Add Delete Button**:
    -   Place a "Delete" button or icon in the top-right corner of each project card (`absolute` positioning).
    -   Use a trash icon (SVG) for visual clarity.
    -   Ensure it has `z-index` higher than the card content but doesn't interfere with the "Enter Project" button.
    -   Add `@click.stop` to prevent triggering the card's navigation event.

-   **Add Confirmation Dialog**:
    -   Create a simple modal component or inline modal structure within `ProjectsList.vue` to confirm deletion.
    -   State: `showDeleteModal` (boolean), `projectToDelete` (Project object or ID).
    -   Content: "Are you sure you want to delete project 'X'? This action cannot be undone."
    -   Actions: "Cancel", "Delete" (red, destructive).

## 2. Logic Implementation
-   **Delete Handler**:
    -   `confirmDelete(project)`: Sets state to show modal.
    -   `handleDelete()`:
        -   Sets `deleting` loading state.
        -   Calls `deleteProject(id)` API.
        -   On success:
            -   Removes project from local `projects` array.
            -   Closes modal.
            -   Shows success feedback (e.g., small toast or alert, or just removing the item is feedback enough, but a transient message is better).
        -   On error:
            -   Shows error message.

## 3. Styling
-   Style the delete button to be subtle (gray) but obvious on hover (red).
-   Style the modal to match existing design (similar to `ProjectsUpload` modal or `UserProfile` modal if any).

## 4. Execution Steps
1.  Modify `src/pages/projects/ProjectsList.vue`:
    -   Import `deleteProject` from API.
    -   Add Delete UI elements to the card.
    -   Add Modal UI at the bottom of the template.
    -   Implement delete logic.
