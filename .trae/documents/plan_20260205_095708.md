## 需求分析
用户希望优化“项目文件结构”的视觉体验，具体要求：
1.  **文件夹展开箭头**：目前默认显示为“下三角”（▼ 或类似），用户希望未打开时显示为“未打开状态的 Icon”（通常是右三角 ▶），展开时才旋转向下。
2.  **对齐 VS Code 风格**：要求文件和文件夹的图标样式与 VS Code 保持一致。
